# 页面职责与导航

这是圆屏端页面落位的主验收标准。

## 系统级入口

- 首页。
- 顶部设置。
- 顶部模式切换：顶部系统栏显示当前用户模式（默认/儿童/访客）。默认对应家长档案池，儿童对应孩子档案池，访客对应临时访客；点按打开模式弹层选择模式。夜间不再作为用户模式，后续如需低亮/勿扰应作为设备设置或 Home Assistant 场景处理。
- 全局 AI 唤醒词。

顶部系统栏元素：时间、首页、页面标题、模式切换、设置。

设置可从顶部系统栏进入，同时也是 BOOT 一级页面循环的最后一站（按 BOOT 从应用页可切到设置，再按一次回到首页）。

初始家庭模式：若设置中没有已保存的模式偏好，且初始化时设备处于离线状态，则初始家庭模式默认为「儿童」；否则为「默认」。已保存的偏好始终优先生效，初始化之后的联网/离线切换不会强制改变当前模式。

## 一级页面

| 页面 | 职责 | 不放什么 |
| --- | --- | --- |
| 首页 | 时间、天气摘要、下一日程、AI/音乐/英语/应用快捷入口 | 深层设置、长列表、复杂内容 |
| 天气 | 温度、湿度、空气、预报、刷新 | 日程事项、设置入口 |
| 日程 | 今日事项、下一项、完成、稍后提醒 | 相册、应用列表、系统诊断 |
| AI | 小智状态、语音对话、AI 动效、工具调用反馈 | 内容库管理、系统设置 |
| 音乐 | SD 音乐、服务器播客、故事音频、播放控制 | 英语练习流程、相册浏览 |
| 英语 | 口语跟读、AI 陪练、学习记录 | 通用资源管理 |
| 相册 | 家庭照片浏览、幻灯片、屏保资源 | 故事、播客、英语课程、小游戏 |
| 应用 | 小游戏、OpenClaw、Home Assistant、NAS 扫描、实验工具 | 设置、天气、音乐、英语、相册、家庭模式 |
| 设置 | 网络、蓝牙、电源、存储、系统、诊断 | 业务应用、小游戏、内容列表 |

说明：本表按职责列出 9 个一级页面，其排列顺序不代表 BOOT 键的实际切换顺序；BOOT 循环顺序见「物理按键导航」。

## 物理按键导航

- 硬件只有两个物理键：电源键（硬件开关机/复位，软件不可编程）和 BOOT 键（GPIO0，唯一可编程物理键）。
- BOOT 单击：正常态循环切换一级页面；启动态进入 WiFi 配网。
- BOOT 长按：切换 AI 语音对话；启动态无动作。
- 底部弧区常驻 AI 按钮：底部系统栏在圆弧区常驻一个 AI 激活按钮，点按切换当前页面 Page Agent 的语音监听（Page Agent First，不做页面跳转），行为与 BOOT 长按一致。
- BOOT 双击（仅在 `CONFIG_USE_DEVICE_AEC` 编译时生效）：空闲态切换设备端 AEC。
- BOOT 一级页面循环顺序（固件实际实现）：首页 → 天气 → AI（页面标题“对话”）→ 音乐 → 相册 → 英语 → 日程 → 应用 → 设置 → 回到首页，共 9 个。
- 说明：AI 页固件内标题为“对话”，相册页的内部枚举名为 `kContent`、标题为“相册”。
- 不在 BOOT 循环内的页面：屏保（空闲约 120 秒自动进入或从相册页启动）、通知、模式（家庭模式）、游戏（小游戏，归入应用）、设置子页（网络/连接/电源/存储/系统/诊断，从设置页进入）；在这些页面按 BOOT 单击会回到首页。

## 默认 Page Agent

一级页面也是 AI 交互上下文。页面内触摸、短指令和语音优先直达当前页面的默认 Agent。

| 页面 | 默认 Agent | 页面内短指令含义 |
| --- | --- | --- |
| 首页 | Home Agent | 继续最近家庭任务 |
| 天气 | Weather Agent | 刷新天气/环境 |
| 日程 | Schedule Agent | 查看下一项、完成、稍后提醒 |
| AI | General Agent | 继续对话或进入 Router |
| 音乐 | Media Agent | 播放、继续、下一集、收藏 |
| 英语 | English Agent | 开始/继续练习 |
| 相册 | Album Agent | 继续幻灯片、进入屏保 |
| 应用 | Tools Agent | 打开工具/小游戏、运行安全家庭动作 |
| 设置 | Device Agent | 查看设备状态、诊断、网络/存储 |

规则：

- 页面提供上下文偏置，不是硬绑定。
- Page Agent 无法处理时可以 handoff 到 Router 或目标 Agent。
- 权限不写死在页面或 Agent 内，统一交给后端 Policy Engine。
- 圆屏不运行 Agent，只传页面上下文和用户输入。

## 四区 UI 基线

- 顶部系统栏：时间、首页、短标题、模式切换、设置。
- 页面主体区：当前页面真正内容。
- 页面操作栏：播放、确认、返回、刷新、扫描等页面动作。
- 底部轻状态栏：单行 toast、同步结果、后端动作反馈；跑马灯（LV_LABEL_LONG_SCROLL_CIRCULAR）内含后端状态令牌，指示后端在线/离线（因顶部系统栏不再显示后端状态）；圆弧区常驻 AI 按钮。

## Remote UI 规则

- Remote UI 不执行脚本、HTML、CSS 或任意远程代码。
- 只渲染白名单组件。
- 只执行白名单 action。
- 不把一级页面伪装成应用项。
- `content` 只作为旧路由兼容，圆屏端正式语义是 `album`。
